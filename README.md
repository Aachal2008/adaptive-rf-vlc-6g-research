# Adaptive RF-Visible Light Hybrid Offloading for Congestion Aware Indoor 6G Networks

## Overview
This repository contains our research paper titled **"Adaptive RF-Visible Light Hybrid Offloading for Congestion Aware Indoor 6G Networks"**, presented at **Multicon EWPPC**.

The paper proposes a hybrid RF–Li-Fi (Visible Light Communication) system to reduce indoor Wi-Fi congestion in future 6G smart classroom environments by intelligently offloading stationary devices to Li-Fi while mobile devices continue using Wi-Fi.
